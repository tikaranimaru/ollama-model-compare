【Lv4: ガントチャート】
プロジェクトのガントチャートを作成してください。左にタスク名の列、右に日付軸（2026-06-01〜2026-06-21の3週間、日単位の目盛り）を取り、各タスクを開始日〜終了日に対応する横バーで描きます。下記データをそのまま使います。

■ タスク（名前 / 開始 / 終了 / 依存）
- T1 要件定義 / 06-01 / 06-04 / なし
- T2 基本設計 / 06-05 / 06-09 / T1
- T3 UI設計 / 06-05 / 06-08 / T1
- T4 実装 / 06-10 / 06-17 / T2, T3
- T5 テスト / 06-16 / 06-20 / T4
- T6 リリース準備 / 06-19 / 06-21 / T5

■ 必須要件
- 日付軸とバーの位置・長さが正確に対応すること（週末の色分けは任意）。
- 依存関係（→）を、依存元バーの終端から依存先バーの始端へ向かう矢印/線で描く。
- バーをドラッグして移動・端をドラッグしてリサイズでき、ドラッグ中はバーの日付位置が更新される。
- バーにタスク名または期間を表示する。

- 評価する点: 日付軸の精密なスケーリング、バー位置の正確さ、依存線の描画、ドラッグ操作の追従。
- 評価しない点: 依存制約の自動再計算、保存、ズーム、クリティカルパス計算。
